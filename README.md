# 📝 Redux Toolkit Todo App

A modern Todo application built using **React**, **Redux Toolkit**, **Tailwind CSS**, and **Vite**. This project demonstrates Redux state management with complete CRUD functionality and persistent storage using Local Storage.

---

## 🚀 Features

- ➕ Add Todo
- ✏️ Edit Todo
- 🗑️ Delete Todo
- 💾 Save Updated Todo
- 📦 Global State Management using Redux Toolkit
- 💽 Persistent Storage with Local Storage
- ⚡ Fast development with Vite
- 🎨 Responsive UI with Tailwind CSS

---

## 🛠️ Tech Stack

- React
- Redux Toolkit
- React Redux
- Tailwind CSS
- Vite
- JavaScript (ES6+)

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/THEmen9/redux-toolkit-todo-app.git
```

Navigate to the project folder

```bash
cd redux-toolkit-todo-app
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```
